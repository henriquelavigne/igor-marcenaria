## 🚀 COMO USAR (3 PASSOS SIMPLES)

### 1️⃣ Descompactar o ZIP
- Baixe o arquivo `igor-marcenaria.zip`
- Clique com botão direito → "Extrair aqui"
- Você terá uma pasta chamada `igor-marcenaria`

### 2️⃣ Testar Localmente
- Entre na pasta `igor-marcenaria`
- Dê duplo clique no arquivo `index.html`
- O site vai abrir no seu navegador!
- Teste o botão do WhatsApp (vai abrir direto no seu WhatsApp)

### 3️⃣ Colocar Online (Hospedar)

Você tem 3 opções **GRATUITAS** e simples:

#### 🌟 OPÇÃO A: GitHub Pages (Recomendado)
1. Crie uma conta grátis em https://github.com
2. Faça upload da pasta `igor-marcenaria`
3. Ative o GitHub Pages nas configurações
4. Seu site fica em: `seuusuario.github.io/igor-marcenaria`

#### 🌟 OPÇÃO B: Netlify (Mais Fácil)
1. Entre em https://www.netlify.com
2. Arraste a pasta `igor-marcenaria` para o site
3. Pronto! Seu site está online em segundos
4. Você recebe um link tipo: `igormarcenaria.netlify.app`

#### 🌟 OPÇÃO C: Vercel
1. Entre em https://vercel.com
2. Faça upload da pasta
3. Site online em minutos
